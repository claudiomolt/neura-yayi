# Neura Yayi — Programa 1h: guerra monetaria mundial

Repositorio privado de investigación y preparación para el programa de 1 hora.

## Tesis

El dólar sigue siendo dominante, pero dejó de percibirse como neutral. BRICS no reemplaza al dólar de golpe: construye redundancia geopolítica. El oro vuelve como neutralidad física. Bitcoin aparece como neutralidad digital soberana.

## Verticales narrativos

1. **Dólar / sanciones / hegemonía financiera** — por qué el dólar manda y por qué se volvió arma política.
2. **BRICS / China / guerra / rutas críticas** — cómo se construyen carriles paralelos: yuan, CIPS, monedas locales, energía y logística.
3. **Oro / Bitcoin / neutralidad soberana** — el regreso de activos que no dependen de permiso; oro para Estados, Bitcoin para individuos y red digital.

## Archivos clave

- `MACHETE.md` — versión extendida para estudiar: narrativa, datos, frases y desarrollo oral.
- `PROGRAMA_1H.md` — escaleta de 60 minutos.
- `SLIDE_STORYBOARD.md` — storyboard visual base.

- `SOURCES.md` — fuentes y claims a verificar/actualizar.

## Regla de trabajo

- La versión extendida puede tener texto, datos curiosos, narrativa y notas.
- La versión simple debe quedar entre **10 y 15 slides**, visual, poco texto, una idea por diapositiva.
- Todo claim fuerte debe tener fuente o quedar marcado como `TODO: verificar`.
